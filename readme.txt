Pour executer l'application:
 - Ouvrir le projet avec IntelliJ Idea
 - Lancer le serveur en exécutant le fichier :
		src/main/java/fr/ensta/launchPattern/MainServeur.java

 - Lancer l'interface utilisateur à l'aide de gradle:
	- Ouvrir le panneau Gradle a droite de l'écran sur IntelliJ 
	- cliquer sur EnstarDesktop > Tasks > application > run.

 Pour les utilisateurs
 - Entrer ses identifiants
 - puis acceder à l'onglet Discussions
 - Actualiser pour voir les autres utilisateurs qui sont en ligne
 - Envoyer un message à un autre utilisateur
 - Recevoir les réponses des autres utilisateurs dans la boite de reception
 - A la fin d'une session, se déconnecter propement revenant à l'onglet qui porte
   votre nom et cliquer sur déconnecter.


Pour l'administrateur
 - Entrer ses identifiants
 - puis acceder à l'onglet Administation
 - Actualiser pour voir les identifiants de tous les utilisateurs
 - Ajouter un nom d'utilisateur et un mot de passe pour creer un nouveau compte
 - Ajouter un nom d'utilisateur et un mot de passe existant pour supprimer un compte
 - Actualiser après chaque action pour voir les changements effectués au niveau utilisateurs
