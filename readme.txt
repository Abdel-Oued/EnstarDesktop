Pour executer l'application:
 - Ouvrir le projet avec IntelliJ Idea
 - Lancer le serveur en exécutant le fichier :
		src/main/java/fr/ensta/launchPattern/MainClient.java

 - Lancer l'interface utilisateur à l'aide de gradle:
	- Ouvrir le panneau Gradle a droite de l'écran sur IntelliJ 
	- cliquer sur EnstarDesktop > Tasks > application > run.

 - Entrer ses identifiants
 - puis accèder à l'onglet Discussions
 - A la fin d'une session, se déconnecter propement revenant à l'onglet qui porte
votre nom et cliquer sur déconnecter.
